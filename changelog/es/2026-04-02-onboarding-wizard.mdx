Los nuevos usuarios ahora reciben un asistente guiado de 4 pasos en lugar de llegar directamente a un dashboard vacío. El sistema de sincronización de partidas también se refactorizó para funcionar por lotes, evitando timeouts en Vercel.

### Asistente de onboarding

Cuando un nuevo usuario inicia sesión por primera vez, se le redirige a un asistente a pantalla completa que le guía por:

1. **Elige tu servidor** — selecciona tu región de Riot desde un desplegable
2. **Vincula tu cuenta de Riot** — introduce tu Riot ID (NombreDeJuego#TagLine) para conectar vía la API de Riot
3. **Preferencias de rol** — opcionalmente elige tu rol principal y secundario (se puede omitir)
4. **Sincronizar partidas** — importa tus 10 partidas clasificatorias más recientes con una barra de progreso en vivo, y después accede al dashboard

### Sincronización de partidas por lotes

La API de sincronización ahora soporta un parámetro `?limit=N`. Cuando se establece, solo se sincronizan N partidas por solicitud y la respuesta incluye cuántas quedan. El hook `useSyncMatches` continúa automáticamente en lotes de 20 con un retraso de 1,5 segundos entre cada uno, para que historiales de partidas grandes no corran el riesgo de alcanzar el timeout de funciones de Vercel.

Después de completar el onboarding, una cookie `sync_on_login` activa la barra lateral para continuar sincronizando las partidas restantes en segundo plano.
