tags: ["improvement"]

La app es ahora más fácil de usar para todos, con mejor soporte para lectores de pantalla, teclados y distintas necesidades visuales.

- **Soporte para lectores de pantalla**: todos los botones con solo icono ahora tienen etiquetas descriptivas, las barras de progreso se anuncian correctamente y todos los elementos interactivos se identifican bien.
- **Colores legibles**: los textos y fondos de toda la app tienen suficiente contraste para leerse bien en ambos temas.
- **Navegación por teclado**: un atajo "Saltar al contenido principal", indicadores de foco visibles en todos los elementos interactivos y todo se puede usar sin ratón.
- **Movimiento reducido**: las animaciones y efectos hover se desactivan automáticamente cuando tu SO está configurado para reducir el movimiento.
