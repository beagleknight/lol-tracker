title: "Tests de humo en CI"
---

Se han añadido **tests de humo con Playwright** que se ejecutan en CI antes de cada merge.

- **18 tests de humo** verifican que todas las rutas de página y endpoints API devuelven 200 — detectan errores en tiempo de ejecución que el build por sí solo no puede encontrar
- Los tests se ejecutan en **modo demo** con datos de prueba, sin necesidad de credenciales externas
- Nuevo comando `npm run test:smoke` para testing local
- CI sube un informe de tests en caso de fallo para facilitar la depuración
