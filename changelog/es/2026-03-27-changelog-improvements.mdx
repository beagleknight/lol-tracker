La página de novedades recibió una buena mejora.

- **Las versiones muestran fechas**: los números de versión ahora indican cuándo se lanzó algo (como 2026.03.9). Más fácil saber qué hay de nuevo.
- **Etiquetas de categoría**: cada entrada está etiquetada como funcionalidad, corrección o mejora con una insignia de color.
- **Filtrar por categoría**: haz clic en una etiqueta en la barra de filtros para mostrar solo ese tipo de entrada.
- **Paginación**: las entradas están divididas en páginas (5 por página) con botones Anterior/Siguiente, para que la página se mantenga rápida a medida que crece la lista.
