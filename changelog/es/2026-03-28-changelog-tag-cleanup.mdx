tags: ["improvement"]

Se reemplazaron las etiquetas de changelog orientadas a desarrolladores con etiquetas amigables para el usuario. La etiqueta "Refactorización" fue eliminada — el changelog ahora usa tres categorías claras: **Funcionalidad**, **Corrección** y **Mejora**.

- **Barra de filtros más simple** — solo tres botones de filtro en lugar de cuatro, facilitando encontrar lo que buscas
- **Etiquetas más claras** — cada entrada del changelog ahora usa terminología comprensible sin necesidad de conocimientos técnicos
