The changelog page got a big upgrade.

- **Version numbers show dates**: version numbers now tell you when something was released (like 2026.03.9). Easier to track what's new.
- **Category badges**: each entry is labeled as feature, fix, or improvement with a colored badge.
- **Filter by category**: click a badge in the filter bar to show only that type of entry.
- **Pagination**: entries are split across pages (5 per page) with Previous/Next buttons, so the page stays fast as the list grows.
